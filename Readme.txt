Реализация управления кораблем (вид сверху)

1)Вместо корабля можно использовать любой спрайт, куб и т.д.
2)Реализовать движение корабля к точке при нажатии левой кнопки мыши. 
3)Реализовать поворот корабля прежде чем двигаться к необходимой точке
4)Реализовать стрельбу ядрами(сферами) в направлении нажатия правой кнопки мыши. 
5)При нажатии клавиши “пробел” реализовать плавное исчезновение корабля. При повторном нажатии плавное появление. Использовать “Unity Coroutines” для реализации 4 пункта.
